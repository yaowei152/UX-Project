DISCIPLINES

In NYP Atheletics, we have 3 different disciplines which includes Sprints, Jumps, Throws, and Distance!

|

Sprints

Running short distances as fast as possible.

Competetive races where speed meets endurance, dash down the track in events such as 100m, 200m, 400m, and hurdles!

Jumps

Achieve the greatest height or distance.

It's a test for athleticism! Leap for height and distance in events like Long Jump, Triple Jump, High Jump, and Pole Vault!

Distance

Test of endurance on the track.

Be physically and mentally strong through distance running. Run distances from 800m to as long as 10,000m!

Throws

Discus, Javelin and Shotput, throw as far as you can!

Where atheletes aim to throw an object as far as possible using Strength, Speed, Technique, and Coordination.
